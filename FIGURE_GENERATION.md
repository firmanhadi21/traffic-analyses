## Generation Workflow

### Step-by-Step Process

#### 1. **Prerequisites**

Ensure you have traffic data GeoPackages:
```bash
ls traffic_jkt_output/*/aggregated_segments.gpkg
ls traffic_bdg_output/*/aggregated_segments.gpkg
ls traffic_smg_output/*/aggregated_segments.gpkg
```

Expected: 24 files (3 cities × 8 periods)

#### 2. **Compute LISA for All Periods**

```bash
python compute_lisa_all_periods.py
```

**Runtime**: ~15 minutes on Apple M1
**Output**: 
- 24 individual LISA GeoPackages in `lisa_results/`
- 3 combined time-series GeoPackages
- LISA cluster maps (Figure 1)
- Supplementary LISA figures

#### 3. **Compute Markov Analysis**

```bash
python compute_lisa_markov.py
```

**Runtime**: ~2 minutes
**Output**:
- All markov figures (Figures 2-6)
- Transition matrices (CSV)
- Analysis report (TXT)

#### 4. **Verify All Figures**

```bash
python verify_paper_figures.py
```

Should output: "✅ SUCCESS: All required figures exist!"

## Figure Specifications for Submission

### Quality Requirements

| Parameter | Setting | Notes |
|-----------|---------|-------|
| **Format** | PNG | For draft; convert to PDF/EPS for final |
| **Resolution** | 300 DPI | Current: ~150 DPI, needs upscaling |
| **Color Mode** | RGB | For digital publication |
| **Width** | 6-8 inches | Single column (6") or full width (8") |

### Current Figure Sizes

All figures are currently saved at default Matplotlib resolution (~100 DPI).

**To regenerate at higher resolution**, modify scripts:

```python
# In compute_lisa_all_periods.py and compute_lisa_markov.py
plt.figure(figsize=(10, 8), dpi=300)  # Add dpi=300
plt.savefig(filename, dpi=300, bbox_inches='tight')  # Add dpi=300
```

## Troubleshooting

### Issue: "File not found" errors

**Solution**: Ensure LISA analysis has been run first:
```bash
python compute_lisa_all_periods.py
```

### Issue: Figures look pixelated

**Solution**: Regenerate with higher DPI:
```python
plt.savefig(filename, dpi=300, bbox_inches='tight')
```

### Issue: Out of memory

**Solution**: Process cities individually by modifying `CITIES` dict in scripts.

### Issue: Missing basemap tiles (if using contextily)

**Solution**: Ensure internet connection or use offline basemaps.
