\textbf{Relevance to the special issue.}
This manuscript directly addresses the special issue's call for open, reproducible tools and
workflows for urban data science. We present a complete, modular, and installable Python
pipeline (\texttt{pip install traffic-congestion-pipeline}) that integrates the HERE Traffic
API, OSMnx, and PySAL into a unified workflow for spatiotemporal traffic congestion analysis.
The pipeline, source code, API documentation, and aggregated datasets are all released openly:

\begin{itemize}
\item \textbf{Source code:} \url{https://github.com/firmanhadi21/traffic-analyses} (MIT licence)
\item \textbf{PyPI package:} \url{https://pypi.org/project/traffic-congestion-pipeline/}
\item \textbf{API documentation:} \url{https://firmanhadi21.github.io/traffic-analyses/}
\item \textbf{Archived datasets:} Zenodo DOI: \href{https://doi.org/10.5281/zenodo.18650759}{10.5281/zenodo.18650759}
\end{itemize}

\textbf{Summary.}
We demonstrate the pipeline across three Indonesian metropolitan cities---Jakarta (metro pop.\ 34M),
Bandung (2.5M), and Semarang (1.8M)---analyzing 264 million traffic observations over 11 months
across 18,694 road segments. The pipeline is city-agnostic: adding a new city requires only a
bounding box and API key.

Our primary empirical finding is that time-of-day ($\eta^2$ = 15--24\%) explains 1,000--4,000$\times$
more traffic congestion variance than any spatial predictor---including network centrality,
POI density, road capacity, and a novel graph-based capacity drop detection metric. Four
independent spatial tests converge on null results, reframing relative congestion as a demand
synchronization problem rather than a spatial infrastructure problem.

\textbf{Contributions.}
\begin{enumerate}
\item A fully open, installable pipeline bridging commercial traffic data with the OSMnx/PySAL
ecosystem---filling a gap identified in recent calls for integrated urban data science tools.
\item A novel graph-based capacity drop detection module using directed OSMnx road networks.
\item The first systematic multi-city comparison of temporal versus spatial congestion predictors
across a 20$\times$ population range.
\item Evidence that normalized congestion metrics inherently remove capacity effects, with
methodological implications for capacity--congestion studies.
\end{enumerate}
